XMLHttpRequest所有流程： let xhr = new XMLHttpRequest(); xhr.open('get','url...',true); xhr.send(); xhr.onreadystatechange = function(){ if(xhr.readyState === 4){ alert(xhr.responseText); } } xhr.open(请求方式， 请求url， 是否异步); xhr.readyState 请求状态值： 0 初始化，还没有调用open()方法 1 载入  已经调用了send() 方法，正在发送请求 2. 载入完成，send()方法完成，已收到所有响应请求 3 正在解析响应内容 4 完成，解析完成，可以在客户端调用
