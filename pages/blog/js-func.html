密封之后 不能添加新属性，不能重新配置，删除任何现有属性  , 但是 可以改属性的值 Object.freeze()   冻结一个对象， 不能添加新属性，不能重新配置，删除属性  , 不能改属性的值 delete obj.name  删除对象的某个属性 Object.defineProperty()的作用就是直接在一个对象上定义一个新属性，或者修改一个已经存在的属性 Object.defineProperty(obj, prop, desc) obj 需要定义属性的当前对象 prop 当前需要定义的属性名 desc 属性描述符 javacript 有三种类型的属性 命名数据属性：拥有一个确定的值的属性。这也是最常见的属性 命名访问器属性：通过getter和setter进行读取和赋值的属性 内部属性：用双括号包围的属性，这不是它真正的名称，而是抽象的 通过Object.defineProperty()为对象定义属性，有两种形式，且不能混合使用，分别为数据描述符，存取描述符 数据描述符：  属性如下： value :             要设置的值  【特有】 writable：      是否可写 【特有】 get：               获取属性 给属性提供getter的方法 set：               设置属性  给属性提供setter的方法 configrable    描述属性是否配置，以及可否删除   false 时，不能删除当前属性 enumerable  描述属性是否会出现在for in 或者 Object.keys()的遍历中 存取描述符 --是由一对 getter、setter 函数功能来描述的属性 对象常量 结合writable: false 和 configurable: false 就可以创建一个真正的常量属性（不可修改，不可重新定义或者删除）  但是 允许再扩展新的属性 禁止扩展 如果你想禁止一个对象添加新属性并且保留已有属性，就可以使用Object.preventExtensions(...) let Person = { name:"ngm" } Object.preventExtensions(Person ) Person.age = 21;
